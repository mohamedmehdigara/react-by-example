# React by Example

## Requirements

- [Node 12](https://nodejs.org/)
- [pnpm 5](https://pnpm.js.org/)

## Running

```sh
# setup
pnpm i

# watch everything
# ignore errors, wait until ready message
# laptop might take off
pnpm start

# or just the gatsby site
pnpm run build-requirements && pnpm run start-gatsby
```
