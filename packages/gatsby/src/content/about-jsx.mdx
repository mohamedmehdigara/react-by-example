# About JSX

- "HTML" that you can write in JavaScript

## JSX

- > JSX is an XML-like syntax extension to ECMAScript without any defined semantics.
  >
  > -- [JSX spec](https://facebook.github.io/jsx/)
- has to be transpiled to JavaScript
- not coupled to React!
- other implementations: [vue], [preact], [inferno], [stencil], [marko], etc...

[vue]: https://vuejs.org/
[preact]: https://preactjs.com/
[inferno]: https://infernojs.org/
[stencil]: https://stenciljs.com/
[marko]: https://markojs.com/

## Examples

- examples of JSX and what it transpiles to under the hood

### Simple

- elements become calls to [`ts∫React.createElement()`][create element]
- first argument: type of element
- third argument: the subtree (aka. [children])

[create element]: https://reactjs.org/docs/react-api.html#createelement
[children]: https://reactjs.org/docs/jsx-in-depth.html#children-in-jsx

```yaml {example}
project: jsx-simple
```

### Props

- second argument: attributes (aka. [props]) as key-value pairs

[props]: https://reactjs.org/docs/components-and-props.html

```yaml {example}
project: jsx-props
```

### Prop types

- props can be of any type, not just strings

```yaml {example}
project: jsx-prop-types
```

### Spread props

- objects can be spread on elements
- applies all keys in the object as props

```yaml {example}
project: jsx-props-spread
```

### Interpolate

- values can be interpolated using `{value}`
- usually renders as `ts∫String(value)`
- `ts∫true`, `ts∫false`, `ts∫undefined`, `ts∫null` never render; allows for [short-circuit logic]
- `<></>` is a [fragment]; allows for rendering multiple elements where one is expected

[short-circuit logic]: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Logical_AND#Short-circuit_evaluation
[fragment]: https://reactjs.org/docs/fragments.html

```yaml {example}
project: jsx-interpolate
```

### Repeat

- use `ts∫array.map()` to "loop" over values
- use the special [`key` prop][key prop] to distinguish elements in the array

[key prop]: https://reactjs.org/docs/lists-and-keys.html

```yaml {example}
project: jsx-repeat
```
